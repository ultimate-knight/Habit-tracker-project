# Next.js + Tailwind CSS Dashboard Skeleton

This project is a simple dashboard skeleton built with **Next.js** and **Tailwind CSS**.  
It features a sidebar, a top search bar, and a main window area, using [Heroicons](https://heroicons.com/) for icons.

---

## Features

- **Sidebar:**  
  Navigation with dashboard, projects, documents, and settings links.

- **Search Bar:**  
  Includes a search icon, notification bell, profile image, and a dropdown for user selection.

- **Main Window:**  
  Central area with a dotted border for your main content.

## Usage

1. **Install dependencies:**
   ```sh
   npm install
   ```

2. **Run the development server:**
   ```sh
   npm run dev
   ```

3. **Open in your browser:**  
   Visit [http://localhost:3000](http://localhost:3000)

---

## Customization

- **Sidebar:**  
  Edit `Sidebar.js` to add or remove navigation items.

- **Search Bar:**  
  Update `Searchbar.js` to change the dropdown options or icons.

- **Main Window:**  
  Place your main content inside `Window.js`.
